 MERN Blog

Full-stack blog application built with MERN stack.

Features
- JWT Authentication
- Google OAuth  
- Blog Posts CRUD
- Comments with likes
- Admin Dashboard
- Dark Mode

 Tech Stack
- Frontend: React, Vite, Tailwind CSS, Flowbite
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Storage: Cloudinary

 Setup

cd api
npm install
node index.js

cd "client side"
npm install
npm run dev
